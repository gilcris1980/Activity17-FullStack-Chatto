# 🟢 ACTIVITY 17: FULL-STACK EXPRESSJS API WITH AUTHENTICATION

### PROJECT_OVERVIEW
A secure full-stack web application featuring user authentication, email verification, and profile management with image upload capabilities. Developed as a requirement for **MSIT114**.

### CORE_FEATURES
* **Secure Authentication**: JWT-based login and registration system.
* **Email Protocol**: SMTP integration for account verification and password recovery.
* **Database Management**: Relational data handling using MariaDB/MySQL.
* **Profile System**: Dynamic profile picture uploads with automated old-file cleanup.
* **Hacker UI**: High-contrast Y2K/Cyber-Pinoy aesthetic for enhanced terminal-style experience.

### TECH_STACK
* **Backend**: Node.js, ExpressJS
* **Database**: MySQL (MariaDB)
* **Security**: BcryptJS (Hashing), JWT (Authorization)
* **Frontend**: HTML5, CSS3, Vanilla JavaScript
